# Paperify - Educational Paper Generation Platform

Educational platform for generating custom exam papers with MCQs, short questions, and long questions.

## Quick Start

```bash
npm install
node index.js
```

Server runs at: http://localhost:3000

## Features

- Custom paper generation
- Multiple question types
- Bilingual support (English/Urdu)
- Payment system integration
- Demo system with free trials

## Tech Stack

- Node.js + Express
- SQLite
- EJS Templates
- Tailwind CSS
